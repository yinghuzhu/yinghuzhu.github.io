打开电商平台的首页后，计算机都做了什么

发表于 2022-04-09

当我们在浏览器地址栏输入电商平台的域名后按下回车到看到首页这段时间内，计算机都做了哪些工作

主要有以下几个过程

浏览器域名解析:

  1. 首先检查浏览器缓存中是否被解析过这个ip地址，如果有解析结束
  2. 如果浏览器缓存中没有命中，会检查操作系统etc文件夹下hosts文件
  3. 如果hosts文件没命中，去本地dns服务器查找
  4. 如果本地dns没明知，开始访问根 dns服务器， 根dns服务器返回本地dns服务器所查询主域名(com, cn 等)服务器ip
  5. 本地dns再发请求到主域名服务器，主域名服务器根据本地dns服务器所在区域返回Name Service地址，这个Name Service就是网站注册的域名服务器
  6. 本地dns再发请求到Name Service地址，Name Service将域名对应的ip地址返回给本地dns
  7. 本地dns缓存这个ip和域名
  8. 本地dns将解析的结果返回给浏览器，浏览器根据TTL值(生存时间值)缓存到本地

发送http请求

  1. 构造http请求，请求包括但不限于各种header, cookie, 语言，国家，idcId(如果有）
  2. 浏览器与服务器经过TCP的三次握手建立连接
  3. 浏览器发送http请求到服务器
  4. 同时异步上报前端用户行为(类似google ga)

服务端入口层

  1. 防火墙作为最重要的安全防护措施，请求进入防火墙，防火墙根据规则判断该请求是否放行; 一些专业的防火墙还可以抗DDOS攻击
  2. 请求通过防火墙后进入负载均衡，类似F5/A10或者软Nginx/HA/lvs等组件；
  3. 根据HTTP header标识判断是否属于此单元请求，如果不是则转发到相应的单元
  4. 负载均衡根据负载均衡策略

路由网关

  1. API 网关 ， 权限检测,流量控制，服务发现，负载均衡，链路跟踪，单元化标识,
  2. 流量分片，请求是否携带了单元化表示，如果没有标识，根据如果有检查是否属于此单元的请求，
